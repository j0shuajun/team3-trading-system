# Team 3 - Ground Rules


### 1. PR 리뷰 및 Approve
- PR을 merge 하기 위해서는 최소 **두 명** 이상의 Approve가 필요하다.
- **PR Template** 내 체크리스트 항목을 확인하고, 형식을 준수하여 작성한다.


### 2. PR Merge
- PR의 Merge는 **PR을 생성(요청)한 사람**이 직접 진행한다.
- 최소 승인 조건(2명 Approve)을 충족한 이후에 merge 한다.
- merge 전 최신 **main branch** 와의 충돌 여부를 확인한다.


### 3. Code Convention
- Code convention은 PyCharm에서 제공하는 기본 Formatter를 사용한다.
- Commit/PR 전 Reformat Code (Ctrl+Alt+L)를 실행하여 스타일을 통일한다.


### 4. Commit Message
- Commit message는 다음 페이지 양식을 준수한다.
- https://wikidocs.net/332862


### 5. Branch Name
- Brance 이름은 **`feature/<기능>`** 형식을 따른다.
